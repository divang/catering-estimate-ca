## 🚀 Deploy in 5 Minutes:

### Step 1: Upload to GitHub
1. Create new repository at [github.com](https://github.com)
2. Name it: `spice-e-zaika-catering`
3. Upload all your project files
4. Make repository public

### Step 2: Deploy to Azure
1. Go to [Azure Portal](https://portal.azure.com)
2. Create a resource → Static Web Apps
3. Use these exact settings:

```
Name: spice-e-zaika-catering
Plan: Free (0 cost)
Region: East US
Source: GitHub
Organization: [Your GitHub username]
Repository: spice-e-zaika-catering
Branch: main
Build Preset: React
App location: /
Output location: dist
```

### Step 3: Go Live! 
Your app will be live at: `https://spice-e-zaika-catering-XXXXX.azurestaticapps.net`
